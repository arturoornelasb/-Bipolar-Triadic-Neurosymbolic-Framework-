# External Datasets Documentation

This project relies on specific external datasets for validation and graph generation.

## 1. GloVe Word Embeddings (glove.6B.50d.txt)
Used for **Appendix D: Real World Validation**.

*   **Description**: Pre-trained word vectors generated from Wikipedia 2014 + Gigaword 5 (6B tokens, 400K vocab, uncased, 50d vectors).
*   **Original Source**: Stanford NLP Group
*   **Download Link**: [http://nlp.stanford.edu/data/glove.6B.zip](http://nlp.stanford.edu/data/glove.6B.zip) (Size: ~822MB)
*   **Installation**:
    1.  Download the zip file.
    2.  Extract `glove.6B.50d.txt` into this directory (`validacion_v1.0.0`).
    3.  (Optional) The script `real_world_glove_validation.py` attempts to automate this if the file is missing.

## 2. Physics Database (final_physics_database.json)
Used for **Graph Topology**, **Romiti Discovery**, and **Appendix C**.

*   **Description**: A JSON database containing physical constants, laws, and concepts.
*   **Location**: Should be present in this directory (`validacion_v1.0.0`).
*   **Source**: Generated by the `ingest_physics_db.py` script (Reverse Engineering) or provided as a static asset.
*   **Repository**: [https://github.com/Kingelanci/graphysics.git](https://github.com/Kingelanci/graphysics.git) (Check here for updates or to download `final_physics_database.json`).
